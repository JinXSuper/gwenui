## Requirements

- Node.js 18+
- Next.js 14+ (App Router)
- Tailwind CSS v4
- TypeScript

## Install via CLI (Recommended)

The fastest way to get started is with the GwenUI CLI:

<PackageManagerTabs>
  <Tab value="npm">
    ```bash
    npx gwen-ui init
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm dlx gwen-ui init
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn dlx gwen-ui init
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bunx gwen-ui init
    ```
  </Tab>
</PackageManagerTabs>

This will guide you through setting up `@gwenui/themes` and configuring your
`globals.css` with GwenUI design tokens.

## Manual Install

### 1. Install themes package

<PackageManagerTabs>
  <Tab value="npm">
    ```bash
    npm install @gwenui/themes
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @gwenui/themes
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @gwenui/themes
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @gwenui/themes
    ```
  </Tab>
</PackageManagerTabs>

### 2. Import tokens in your globals.css

```css
@import "@gwenui/themes/tokens";
@import "@gwenui/themes/fonts";
```

### 3. Add a block

```bash
npx gwen-ui add pricing
```

> Blocks are copied directly into your project — you own the code.
