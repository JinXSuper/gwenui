# Changelog

All notable changes to GwenUI are documented here. Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.4.1] — 2026-05-22

### Added

- **Bento Grid** — asymmetric feature grid with staggered Framer Motion entrances
- **Feature Walkthrough** — step-by-step product tour block
- **Comparison Table** — tiered feature comparison with responsive layout
- Preview routes for in-browser block inspection
- Agent skills under `gwenui/skills/` (animation, R3F, docs writing, landing hierarchy)

### Changed

- Docs site navigation and block registry alignment
- Monorepo tooling: pnpm 10, Node 22, Vercel/Corepack deploy config

---

## [0.3.0] — 2026-04-00

### Added

- **AI Chat Hero** — LLM-ready chat layout with suggestion chips and sidebar
- **ArticleCard** and **ImageCard** block primitives (docs/registry)

### Changed

- AI block category metadata and CLI registry entries

---

## [0.2.0] — 2026-03-00

### Added

- **Before-After Slider** — draggable image comparison
- **Meet the Team** — team grid with hover states
- **Dynamic Pricing** — three-tier pricing with motion
- **Testimonial Marquee** — infinite scroll social proof

### Changed

- Expanded `gwenui-blocks` package exports
- Theming docs for OKLCH tokens

---

## [0.1.0] — 2026-02-00

### Added

- **Parallax Hero** (Supreme) — R3F/WebGL hero with floating parallax and text effects
- **`@gwenui/cli`** published — `gwenui` bin with `init`, `add`, `list`, `diff`
- **`@gwenui/react`**, **`@gwenui/supreme`**, **`@gwenui/themes`** package scaffolding and docs
- GwenUI docs site (`apps/web`) on Next.js
- Block registry and `block.json` metadata format
- MIT license path for basic blocks via [LICENSE-MIT](./LICENSE-MIT)
