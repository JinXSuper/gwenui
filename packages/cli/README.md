# @gwenui/cli

Interactive TUI installer for GwenUI block registry.

## Install

```bash
npm install -g @gwenui/cli
```

## Usage

```bash
gwen-ui
```

Browse and install 2D blocks (`@gwenui/react`) and 3D supreme blocks
(`@gwenui/supreme`) directly into your project.

> Coming Soon — full block browser launching with GwenUI v1.0
