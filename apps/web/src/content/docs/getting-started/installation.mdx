## Requirements

- Node.js 18+
- Next.js 14+ (App Router)
- Tailwind CSS v4
- TypeScript

## Install via CLI (Recommended)

The fastest way to get started is with the GwenUI CLI:

<PackageManagerTabs command="dlx gwen-ui init" />

This will guide you through setting up `@gwenui/themes` and configuring your
`globals.css` with GwenUI design tokens.

## Manual Install

### 1. Install themes package

<PackageManagerTabs command="add @gwenui/themes" />

### 2. Import tokens in your globals.css

```css
@import "@gwenui/themes/tokens";
@import "@gwenui/themes/fonts";
```

### 3. Add a block

```bash
npx gwen-ui add pricing
```

> Blocks are copied directly into your project — you own the code.
